# Consumer Complaint Analyzer

## Overview

Consumer Complaint Analyzer is an NLP-based Machine Learning application that predicts the category of financial consumer complaints.

## Features

- Complaint Prediction
- Bulk CSV Prediction
- Dashboard
- Download Results
- Streamlit UI

## Technologies

- Python
- Streamlit
- Pandas
- Scikit-Learn
- NLP
- TF-IDF

## Dataset

Consumer Financial Protection Bureau (CFPB)

## Run

```bash
pip install -r requirements.txt

streamlit run app.py
```
